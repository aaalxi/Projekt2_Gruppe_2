Members.txt fil indeholder data i følgende ordre: 
[1] Bruger ID : String
[2] Navn : String
[3] Fødselsdato : LocalDate
[4] Er de konkurrencesvømmer? : boolean
[5] Restancebeløb : double
[6] Træner knyttet til medlem : String
[7] Discipliner knyttet til medlem : Enum
[8] Dato medlem blev tilføjet til system : LocalDate
[9] Næste betalingsdato for medlem : LocalDate
[10] Er medlem aktiv eller passiv : boolean

CompetitionResults.txt indeholder data i følgende ordre:
[1] Dato for resulatat : LocalDate
[2] Navn på svømmer : String
[3] Holdet for svømmeren (U18/O18) : String
[4] Disciplin for stævnet : Enum
[5] Meter distance for heat : double
[6] Minutter for heat : double
[7] Er de konkurrencesvømmer? : boolean
